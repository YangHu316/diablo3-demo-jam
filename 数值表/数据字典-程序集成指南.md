## 4. 关键不变量 / 口径红线(别破坏)

1. **伤害 5 乘区独立**,严禁合并预计算。
2. **掉落 drop_table 每行 5 稀有度权重和必须 = 100**。
3. **传说获取主通道 = loot_timeline 保底**(脚本必掉+软/硬保底);drop_table 的「传说%」是基础 roll,**勿与保底双算**(精英行=精英橙率 3/5)。
4. **白怪几发死 = 持续DPS口径**(非利箭单发);单发开局 L1-3 取 4-5 发属未成型爬坡。
5. **史诗白值压在 Tier 区间内;传说=上沿×1.1**(刻意越级,不受 player_growth 区间约束)。
6. **承伤红线**:满血最短死亡≥1.5s;单次≥50%生命必有≥0.8s预警。
7. **品质**:核心 3 档(魔法/稀有/传奇);**5 档为 opt-in 扩展**,代码默认 3 档别误开。
8. **装备槽中文 = `EquipSlots.SLOT_DISPLAY`**(amulet=项链,已全表对齐);程序映射用 **slot id**(head/amulet/ring/bow/quiver…)非中文。

## 5. 程序集成指南

### 5.1 读取 CSV(最小示例)
```gdscript
# 直接解析源 CSV(不依赖 Godot 翻译导入)
func read_csv(path: String) -> Array:
    var rows: Array = []
    var f := FileAccess.open(path, FileAccess.READ)        # path 如 "res://数值表/affixes.csv"
    var headers := f.get_csv_line()
    while not f.eof_reached():
        var cols := f.get_csv_line()
        if cols.size() <= 1 and (cols.is_empty() or cols[0] == ""): continue
        var d := {}
        for i in headers.size(): d[String(headers[i]).strip_edges()] = (String(cols[i]) if i < cols.size() else "")
        rows.append(d)
    f.close()
    return rows
```
> **导入注意**:本目录多数 CSV 被 Godot 默认按 **翻译(Translation)** 导入(产生 `.translation`);掉落物 3 表(`base_items`/`epic_items`/`affix_naming`)已设 **keep 导入(`.csv.import` importer="keep")不产翻译**。无论哪种,**用 `FileAccess` 读源文件**最稳;若想全部不产翻译,把各 `.csv.import` 设为 `importer="keep"`。

### 5.2 怪物结算
```
某原型在 L 级:
  生命 = monsters.csv[L].白怪生命  × enemy_types[原型].生命系数
  攻击 = monsters.csv[L].白怪攻击  × enemy_types[原型].攻击系数
  （蓝/黄/骸骨成品列已在 monsters.csv 算好,可直接取）
  行为/技能/移速/距离/前摇 = enemy_types[原型]
  精英(蓝×2.5/黄×5)作用于白怪基准,不与原型系数叠乘
```

### 5.3 掉落结算
```
1. 取掉落源 → drop_table[源]:掉落率→是否掉;件数;按 5 稀有度权重 roll 稀有度
2. 稀有度 → rarities[稀有度].词缀条数;gear_by_rarity[稀有度][Tier].武器DPS/护甲(无售价·经济已移除)
3. 部位 → base_items 基底(普通/精良/稀有)；史诗→epic_items 抽一件(保送其特效词缀=affixes T3顶 + 其余按部位随机)；传说→legendaries
4. 词缀 → affixes(按 base_items/affixes 的部位约束 roll);命名 → affix_naming(精良前缀)/固定名(史诗/传说)
5. 传说额外走 loot_timeline 保底(首橙≤8min、屠夫/守门必掉…)
```

### 5.4 品质 3→5 档扩展(开启 5 档需改代码 —— 系统②域)
- `scripts/data/item_instance.gd`:`Quality` 枚举 `{MAGIC,RARE,LEGENDARY}` → 加 `COMMON,EPIC` + `QUALITY_NAMES/COLORS`(普通灰/史诗紫)。
- `scripts/systems/drop_system.gd`:品质权重 3 元 → **5 元,读 `drop_table.csv`**(旧 3 元作废);按「原型键」join `enemy_types`。
- `scripts/systems/item_generator.gd`:`AFFIX_COUNT` 补 `COMMON:[0,1]/EPIC:[4,5]`;实现史诗「保送 1 条顶值词缀」;白值读 `gear_by_rarity`。
- `data/monsters.tres`(**外部运行时仓库**):`skeleton_guard.health_mult` **1.6→1.0**(骸骨修正)。
