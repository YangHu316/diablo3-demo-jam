## 这一版基于的设计(V2.1 当前口径)

- 职业:单职业**亚马逊·弓系**,主属性=敏捷,资源=**专注**(上限100/被动回复8每秒/利箭命中回4)。
- 技能:**5 个主动**(利箭/多重射击/冰冻箭/翻滚回避/女武神)+ 符文 + **被动 5 选 2**;技能槽 5,第 5 级全开。
- 等级:**1–8**,经验 `XP_to_next(L+1)=XP_to_next(L)×1.6`,L1=300,累计到 8 级 ≈12922。
- 伤害:**5 乘区** `武器均伤 × 技能倍率% × (1+敏捷/100) × 暴击结算 × (1+元素伤%)`。
- 怪物:**白怪生命=40×1.31^(L-1) / 攻击=15×1.17^(L-1)**;蓝名×2.5生命/×1.3攻;黄名×5生命/×1.4攻。
- 白怪 **6 种**(走尸/疯犬/弓手/肿胀者/召唤者/**骸骨卫士盾兵**);已删诈尸走尸、掘地食腐者。
- 唯一 Boss = **屠夫**(生命≈11000/攻≈75);中段教学战 = 黄名"地穴守卫"(非 Boss)。
- 传奇 **5 件**;词缀池 **10 条**;品质 **3 档**(魔法/稀有/传奇);首橙 **≤8min**。
- 终局 = **梦魇 2 层**(纯数值复用关卡 2/3 Tile)。

## 关键公式速查

- 伤害(单次)= `武器均伤 × 技能倍率% × (1+敏捷/100) × 暴击结算 × (1+元素伤%)`;`期望DPS = 上式 × APS`(APS 线性折算,非乘区)。
- 生命上限 = `40 + 10×等级 + 10×体能`。
- 护甲减伤 = `护甲/(护甲+20×攻击者等级)`。
- 白怪生命 = `40×1.31^(L-1)`;白怪攻击 = `15×1.17^(L-1)`;白怪经验 = `8×1.45^(L-1)`(怪物等级动态同步玩家,8 封顶)。
- 骸骨卫士 = 白怪 ×1.6 生命 / ×1.3 攻 / ×1.5 经验 / 移速 70%;正面 120° 弧非穿透箭伤 −75%。

## 本次"重新优化"做了什么

1. **整体改写到 V2.1 系统**:旧版用的是"普通射击/强力射击 + 扁平血量 + 冷却制 + 小经验值"——与你这套**专注/利箭/XP×1.6/指数血量/5 乘区**完全不兼容,已废弃。本套 100% 按 05/附录B 重出。
2. **掉落模型换成保底驱动**:旧版是随机稀有度权重;本套按 V2.1 的**脚本固定 + 软/硬保底 + 首橙白名单**时间轴(见 `loot_timeline.csv`)。
3. **怪物名单对齐**:6 白怪(含骸骨卫士盾兵),删除诈尸/掘地;`monsters.csv` 含骸骨卫士派生列。
4. **修正 D1**:屠夫攻系数由错误的"×约5"改为"**×约2(攻≈75 锁定)**",`05 §8` 与 `附录A` 审计同步标记为已解决。
