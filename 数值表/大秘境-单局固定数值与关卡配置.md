## 〇、设计原则(本版铁律)

1. **5–10 分钟快节奏割草**:无双手感优先,白怪 1 发秒、精英几秒、守门人 45–75s。
2. **取消等级成长**:删 XP/升级对玩法的驱动。怪物用**固定绝对数值**(不再随等级缩放);玩家**开局即中期满装**,面板写死。
3. **固定一切**:玩家一套**固定装备 + 固定数值技能**,全程**不可更换/不可切符文**;键位**严格参照《暗黑破坏神3》**。
4. **掉落极简**:仅大秘境 L2;**小怪零掉落**;**守门人死 → 固定爆装(满地橙 + 1–2 绿套装)**;**严禁橙/绿以外品质**。
5. **地图极简**:移除所有传送门与宝箱,仅留入口;移除小关卡 L1。

> 与既有工作的衔接:经济已移除(沿用);经验提速曲线 / L2 经验预算**本版整体作废**(无等级);5 档稀有度体系**收窄为仅橙+绿掉落**(白/蓝/黄/紫不再作为掉落出现,仅作历史档保留);复用 L2 大地图 + 八边形守门人房 + 已有 HUD/背包/掉落/物品系统。

**进度球机制(取代精英击杀直接权重)**:
- 精英死亡 → 在尸体位置生成 N 颗**进度球**(蓝名 1 / 黄名 2);玩家**走近自动吸取**,飘字「**+5% ▲进度**」。
- **每颗进度球 = +5% 大秘境进度条**(≈ +5.3 进度点·RiftManager GOAL=106)。蓝名 +5% / 黄名 +10%。

**实现分工**:战斗①(精英实体:`.tres` HP/ATK + 模型缩放 + 颜色辉光 material + 词缀行为 + 大前摇 + 预生成/aggro + 死亡触发掉球);系统②(进度球场景 + 拾取 +5% → `RiftManager.add_progress` + 精英直接权重置 0);美术(蓝/金辉光 shader + 进度球发光 VFX + 拾取特效)。

## 五、平衡校验摘要

- **TTK 单调**:白怪150 < 利箭365 → 1 发;蓝名375 → 2 发;黄名750 → 3 发;守门人24000 / 547 ≈ 44s 纯利箭,叠 P2 无敌窗 + 烧地板走位 → 45–75s。**单调递增,无倒挂**。
- **专注经济**:利箭 0 耗 + 命中回 4 + 被动 8/s;多重 20 / 冰箭 25。利箭连射不卡蓝,多重/冰箭穿插可持续(回蓝 > 消耗节奏)。守门人战站桩利箭为主、多重泄能,无断蓝。
- **生存**:玩家实有效血池 ≈ 1200 / 0.5 减伤当量 ≈ 面对白怪极宽容;守门人靠走位(烧地板/冲锋/肉钩)而非血量墙。
- **割草密度**:≈106 进度权重 / ~6min(沿用 10 号 §6.3 表),留时间球 +3×N 安全垫。
- ⚠️ **守门人 24000 HP 与 TTK 为起始值**,Day2 由关卡C playtest 微调(优先调 boss HP,不动玩家面板)。
