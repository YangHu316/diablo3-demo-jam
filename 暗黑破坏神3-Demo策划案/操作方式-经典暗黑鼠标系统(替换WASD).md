## 一、核心原则(经典暗黑三键合一)

经典暗黑没有 WASD——**移动、攻击、施法全部用鼠标**。三件事压在两个键上:

| 输入 | 行为 |
|---|---|
| **左键(LMB)** | ① 点**地面** → 寻路走向该点(按住=持续朝光标走)。② 点**敌人** → 走进射程并用「左键技能」攻击(按住=持续攻击)。**移动与主攻合一**,这是暗黑手感的根。 |
| **右键(RMB)** | 释放「右键技能」朝光标方向(按住=持续)。 |
| **Shift + 左键 / 按住 Shift** | **原地站立攻击**(脚不动,朝光标射)——弓系风筝/站桩的关键键(经典暗黑 "Stand Still")。 |
| **强制移动键**(默认 `空格` 或可绑) | 朝光标移动且**无视敌人**(不自动攻击),纯走位(经典暗黑 Force-Move)。 |

**取消后摇(继承红线)**:任何攻击/技能 Recovery 阶段,一收到移动/换技能输入立即打断、0.1s 融合;禁止"必须播完"的射击后摇——这是风筝可靠性的地基。
**无 miss / 翻滚穿怪 / 利箭穿透**:继承不变。

## 二、技能指派(对齐 Demo 的 5 技能 / 5 槽)

Demo「技能槽 5 = 左键/右键 + Q/W/E 三键(原 1/2/3 → QWER·暗黑3风格)」,移动从 WASD 改成左键点地:

| 键 | 默认技能 | 说明 |
|---|---|---|
| **左键 LMB** | **利箭**(产能/主攻) | 点地走、点敌射;零消耗,永远可按(死锁保险) |
| **右键 RMB** | **多重射击**(泄能) | 朝光标扇形 |
| **Q** | 冰冻箭(控场) | 朝光标 |
| **W** | 翻滚回避(位移) | 朝**光标反方向后撤** / 或朝移动输入方向;0.4s 无敌帧 |
| **E** | 箭雨风暴(引导/旋转箭雨 AoE·招牌) | 按住引导旋转,每 0.3s 向四周 360° 射一轮箭雨 |
| **R** | (空槽·预留) | 本版仅 5 技能,第 4 技能位留空 |

> 键位:左键/右键/Q/W/E(+ R 预留)。**V3.0 单局版技能固定**,不可更换/不可切符文(原"自由拖拽换键"在固定版停用)。

## 三、其它(经典暗黑 QoL)

- **小地图**:`Tab` 切换全/角;**背包**:`I`/`B`;**角色面板**:`C`。
- **拾取**:踩上自动名牌,**点击地面物品**拾取(经济已移除,无金币/血球自动吸附仍保留血球)。
- **药水**:免费、默认 `1`(原 D3 在 `Q`;Q 改技能故移至数字 1·可绑,暗黑1/2 腰带风格),不进数值循环。
- **手柄**:不做(继承)。
- **鼠标悬停**:精英名条+词缀名(P0)、敌人描边。

## 四、与旧表述的差异(本文取代)

| 旧(00~09/分工/designs) | 新(本文) |
|---|---|
| WASD 8 方向移动 | **左键点地寻路移动**(无 WASD) |
| 鼠标朝向(角色面朝鼠标) | 角色面朝**移动/攻击目标**;施法朝光标 |
| Shift 强制站立 | **Shift 原地攻击**(站桩射击) |
| Force Move | **强制移动键**(无视敌人走位) |

## 五、给战斗①(`scripts/entities/player.gd` 改造交接)

当前 `player.gd` = WASD(`move_left/right/up/down`)+ 鼠标地面投影朝向 + 左键射箭。改为经典暗黑:

1. **移动**:删 WASD 8 方向;改 **左键点地 → 目标点**,玩家用 `NavigationAgent3D` 寻路过去(或在开阔白盒里直接朝目标点 `move_toward`);`SPEED=7` 不变。按住左键 = 持续更新目标点为光标。
2. **左键合一**:射线命中**敌人** → 设为攻击目标,走进 `利箭` 射程后施放(面向目标);命中**地面** → 设为移动目标。
3. **右键 / Q / W / E**:触发 `SkillSlotManager` 对应槽,朝光标方向(`SkillExecutor` 已按光标/朝向出箭,改取"光标方向"为准)。
4. **Shift**:按住时**不移动**,左键=原地朝光标射(站桩);松开恢复点地走。
5. **强制移动键**:朝光标移动且**不设攻击目标**(无视敌人)。
6. `skill_executor.gd:100` 现"优先 WASD 输入取位移方向"——改为**取光标方向 / 当前移动目标方向**(无 WASD 后)。
7. 取消后摇逻辑保留;翻滚 `W` 键改为"朝光标反方向后撤 0.4s 无敌"。
> 输入映射(`project.godot` InputMap):移除 `move_*`;新增 `force_move`(空格)、`stand_still`(Shift,或直接读 `Input.is_key_pressed(KEY_SHIFT)`)、`potion`(**1**)。左/右键用 `attack_primary`/新增 `attack_secondary`;**技能键 `skill_1/2/3` 重绑到 `Q/W/E`**(原 1/2/3),`skill_4`=R 预留。键位事实源 = `数值表/player_keymap.csv`。
