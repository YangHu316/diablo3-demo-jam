## 2. 美术总立场与游戏基调

**一句话基调:阴郁哥特地下城里的爽快割草。** 新崔斯特姆的灰败、受难大教堂的烛火霉味、痛苦大厅的血腥,作为压抑的底色;亚马逊用箭雨把成群亡灵碾碎的远程爽感 + 掉橙多巴胺,作为释放面。设定越压抑,战斗与掉落的释放越爽——这是暗黑情绪核心,也是 Demo 验证目标 V1(弓系手感)/V2(掉落兴奋)的情绪载体。

- **空间情绪曲线(对齐 03 §9 密度曲线)**:城门外谷段(压迫铺垫)→ 尸潮/大教堂峰段(割草释放)→ 掉落光柱(奖赏信标)→ 地穴守卫与盾墙(战术紧张)→ 屠夫高压 → 梦魇橙雨(宣泄)。
- **光的叙事**:幽暗烛火被战斗闪光与掉落光柱点亮;**传奇橙光柱 = 幽暗中的希望信标**(首橙 ≤8 分钟是必须观测的兴奋峰值,见 03 §3 / 04)。
- **分级基调**:风格化黑暗——有血、有亡灵但不写实血腥(肢解/血液贴花为 P1,血腥可开关,见 03 §3),既可读、可买、又分级友好。

## 3. 艺术风格

**风格定位:风格化低多边形 / 轻手绘的黑暗哥特奇幻(painterly low-poly),3D 模型做 2.5D 表现。**

为什么不是写实:**全部自购 + 2.5D 俯视**两个约束直接锁死答案——资产店对"低多边形哥特"供给最足、性能最轻;2.5D 俯视距离下靠剪影 + 颜色 + 自发光读信息,而非贴图细节。

| 维度 | 方向 |
|---|---|
| 形体 | 厚重、剪影清晰、顶面有辨识度;少做俯视看不见的小细节 |
| 相机 | 固定角度俯视/微等距(Godot 3D 场景 + 角度相机),玩法在地面平面 |
| 光照 | 火把/火盆/灵魂之瓮点光 + 雾 + 暗角;危险物/掉落/精英光环用**自发光**保可读;烘焙为主 + 少量动态光(性能) |
| 色彩 | 冷色低饱和打底 + 暖色点缀 + 稀有度色 + 危险红;**高明度对比**是俯视可读性关键(见配色板) |
| VFX | 风格化平涂(实现规格见 03 §3);血液风格化、不写实 |
| 参考意象 | 哥特大教堂废墟、烛火地穴、被诅咒的崔斯特姆村;"一人碾过亡灵潮"的力量幻想(意象参考,做原创化造型,不照搬具体作品) |

**配色板(hex,美术直接用):**
- 基底冷:石灰 `#6B6F76`、墨蓝黑 `#14171F`、暗砖 `#3A2E2A`、苔绿 `#4A5A43`
- 暖色点缀:火光橙 `#FF8A3D`、烛黄 `#FFC95E`、金 `#D4A24E`
- 稀有度(3 档 + 白底装):普通白 `#E8E8E8`、魔法蓝 `#4FA3FF`、稀有黄 `#FFD23F`、**传奇橙 `#FF7A1A`(永远最亮最饱和)**
- 元素/状态(对齐 03 命中粒子分元素 + 4 精英词缀):物理灰白 `#E8E8E8`、火焰 `#FF5A1F`、冰冻青 `#5FD0E0`(冰冻箭/冰冻词缀)、毒/腐 `#7CCB3A`、流血暗红 `#8E1B1B`、危险/预警红 `#FF3B30`(熔火/烧地板/肉钩贴花)

**颜色读取原则(可读性骨架)**:亚马逊最亮最干净 → 随从科尔玛暖色/圣光 → 白怪晦暗冷色 → 精英橙光环 → 屠夫巨大+血红 → 掉落按稀有度色 → 危险/预警=红。每样东西靠颜色+剪影一秒可辨。

## 5. 资产清单(按类别)

### 0) 采购策略(置顶,先读)
- **选 1 套环境 kit + 1 套角色/动画体系,风格统一**;混风格是最大美术翻车点。
- 人形角色尽量买**标准人形骨架(humanoid / Mixamo 兼容)**网格,动画走 **Mixamo(免费、标准人形、海量动作)**,压低采购与重定向成本;非人形(犬/肿胀者/巨型屠夫)买自带动画的包。
- 风格化低多边形供应商家族(起点,自行核实授权/内容):Synty POLYGON(Dungeon/Fantasy Kingdom/Knights)、KayKit(Dungeon/Adventurers)、Quaternius、Kenney。
- VFX 从 Unity/UE 迁 Godot 较麻烦,优先直接面向 Godot 的素材或通用粒子/贴图;反馈实现规格以 03 §3 为准。
- **每件采购前确认:① 风格匹配 ② 授权允许商用 ③ 含所需动画 ④ 字体含中文字形。**
