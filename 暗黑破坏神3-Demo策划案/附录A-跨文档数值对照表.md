## 跨册不一致(Discrepancy)清单

经全量核对,**绝大多数关键数值(经验曲线、属性/生命、武器Tier、5技能倍率、怪物HP表、屠夫HP≈11000、掉落时间轴/首橙≤8min、梦魇×1.5/×2.2缩放、TTK目标)在 7 个分册间完全一致**。发现的问题如下:

### ✅ D1(原真实数值冲突 — V2.1 已修正)
- **位置**:`05-Demo成长与数值` §8 line 282 vs §8 line 295(及全分册"屠夫攻≈75"口径)。
- **问题**:line 282 的怪物系数说明写"屠夫 **×约55 生命 / ×约5 攻**(锁定值)"。其中"×约5 攻"算出 15×1.17⁶×5 ≈ **192**,与同段 line 295"攻击≈75"以及 00/01/03 全部锁定的"屠夫攻≈75"**自相矛盾**。屠夫攻击 75 对应白怪攻@7(≈38)的倍率应为 **约×2(×1.95)**,而非×5。生命侧"×约55"正确(无需改)。
- **处置(V2.1 已完成)**:已将 05 §8 的"×约5 攻"改为"**×约2 攻(攻≈75 锁定值)**",与 line 295 及其余分册口径统一。同步在 `data/constants.csv` 的 `屠夫攻击` 行注明"D1 已修正:非×5"。

### 🟡 D2(表述/取整差异 — 非冲突,建议统一以免误读)
- **位置**:`05` §4.1 "目标用时(分钟·中位)"列 vs 总案/SPEC 同名"目标用时(分)"列。
- **问题**:05 填**逐级单段时长**(2.5/3.0/3.5/4.0/3.0/2.5),SPEC 填**累计到达区间**(0~2.5 / 2.5~5.5 …)。两者可换算、口径一致,但**同名列含义不同**,直接对照易误读为冲突。
- **建议**:把 05 §4.1 该列改名为"本级用时(单段)"或补一列"累计到达时刻",与 SPEC 区分清楚。注:该差异不影响任何下游锚点(L2≈2.5…L8≈24 四处一致)。

### 🟡 D3(口径范围差异 — 非冲突,记录备查)
- **武器值列**:04 §3.3 给"白值掉落区间"(T2=9~18/T3=20~32),05 §5.2 给"标准体武器均伤"(含装备加成的点值,如Lv4=14/Lv7=25)。两者是"掉落区间"与"装配后点值"的不同口径,标准体点值落在对应区间内,**非冲突**。
- **密度列**:06 §4.1 梦魇层密度(14~18 / 18~24)与 03/01 主线密度(8~14/屏,峰值15~20)分属不同区段,**非冲突**。
