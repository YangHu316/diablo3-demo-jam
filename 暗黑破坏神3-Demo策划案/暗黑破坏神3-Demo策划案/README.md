# 《暗黑破坏神3》垂直切片 Demo 策划案

本目录是用于**快速验证**的垂直切片(Vertical Slice)Demo 策划案,基于完整版策划案(见 `../暗黑破坏神3策划案/`)裁剪与改编而成。

> 文档版本:**V2.0** ｜ 日期:**2026-06-13**
> 本 README 为全组索引与**单一事实源(口径速览)**。各分册数值/设定如与本表冲突,以分册正文中标注的权威口径为准;跨分册口径冲突一律以"全组统一约束"为准。

## Demo 一句话定位

单机 Demo,总时长**约 30 分钟**(主线 20min 到屠夫 + 梦魇挑战 10min/2 层),用第一幕精简剧情 + **单职业亚马逊(弓系远程)** + **1~8 级压缩成长**,验证两件事:**①弓系远程核心战斗手感是否"爽"(含职业内不同 Build 手感差异);②"掉落→构建→挑战更难内容"循环能否在 30 分钟内成立。**

## 全组统一约束(总纲速览)

| 维度 | Demo 约定(V2.0 权威口径) |
|---|---|
| 总时长 | 约 30min = 主线 20min(到屠夫)+ 梦魇挑战 10min(2 层) |
| 剧情范围 | 第一幕精简版,**主线 3 个关卡场景块**:新崔斯特姆 → 受难大教堂(救迪卡·凯恩)→ 痛苦大厅·屠夫 Boss 收尾;支线大幅砍除 |
| 职业 | **仅亚马逊(弓系远程)**,主属性=敏捷,资源=专注(回复-爆发型,上限 100,被动回复 8/秒,利箭命中回 4) |
| 等级 | **1~8 级**,每级必有新解锁;**技能槽=5**(左键/右键 + 3 数字键),第 5 级即全部开放;不存在第 6 槽 |
| 技能 | 5 个主动技能(利箭/多重射击/冰冻箭/翻滚回避/女武神)+ 符文系统 + 被动(5 选 2) |
| 掉落 | **首件传奇 ≤8 分钟**(脚本软/硬保底);**传奇 5 件**;**词缀池 10 条**;**品质 3 档**(魔法/稀有/传奇);获取仅"掉落+保底";智能掉落 100% 定向敏捷 |
| 终局 | 通关屠夫后解锁"梦魇挑战"——**2 层、纯数值复用**(复用关卡 2/关卡 3 的 Tile,仅做数值缩放+精英/密度调整,不做新地图/新 Boss/新美术) |
| Boss | 全 Demo **唯一 Boss = 屠夫**;原骷髅王 Boss 取消,中段教学战职能由黄名精英"地穴守卫"承担(非 Boss);梦魇 2 层守门"梦魇屠夫"=屠夫数值调整复用 |
| 砍掉 | 制造经济、巅峰、赛季、多人社交、交易;随从仅圣堂武士科尔玛(简化:跟随+普攻+10 秒治疗);图鉴/成就=加分项(P2,不作验证依赖) |
| 节奏原则 | 等待/跑路/成长间隔压缩;**每 2~3 分钟一次"新东西"** |

## 阅读顺序建议

- **评审/决策视角**:00 → 08(先看验证什么、怎么判定),再抽查 01/02/04。
- **开发执行视角**:00 → 01(流程)→ 02/03(战斗向)→ 04/05(数值向)→ 06 → 07,08 交给程序与用研对接埋点。
- **数值平衡视角**:05(成长/伤害/TTK/梦魇缩放)为权威值底座,04(掉落/词缀/传奇)、02(技能倍率/资源)、06(梦魇缩放)均与其对齐。

## 与完整版的关系

每份 Demo 文档均含"**正式版 vs Demo 对照表**"与裁剪理由;本 Demo 是一条聚焦"亚马逊弓系手感 + 掉落构建循环"的 30 分钟垂直切片,刻意收口为单职业、1~8 级、5 技能槽、唯一屠夫 Boss、梦魇 2 层数值复用。Demo 验证通过后,各系统按完整版策划案(`../暗黑破坏神3策划案/`)扩展回全量设计(多职业、1~70 级、6 技能槽、全幕剧情与多 Boss、独立终局秘境等)。
