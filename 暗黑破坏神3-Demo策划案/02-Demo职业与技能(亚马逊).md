## 1. 设计目标与亚马逊体验定位

### 1.1 本文档要回答的三个问题

1. **节奏**:正式版 1~70 级的解锁内容,如何压进 **1~8 级 / 主线 20 分钟 + 梦魇 10 分钟**,且每级都有"新玩具"?
2. **取舍**:一个职业 20~25 个技能 ×5 符文的正式版内容池,砍到"**5 主动 ×1~2 符文 + 5 选 2 被动**"后,凭什么还能拼出 **2~3 套手感不同的弓 Build**?
3. **闭环**:专注(回复—爆发型)这套资源循环,在没有毕业装备词缀(APoC 类)兜底的 **8 级框架**下如何成立?

### 1.2 单职业聚焦:亚马逊弓系远程爽感(Demo 验证的第一命题)

V1.0 用两个职业做"近战 vs 远程"的极端采样;V2.0 收口为**单职业亚马逊**,把全部验证预算压在**一条远程弓系手感曲线**上做深、做透。验证命题从"两职业手感差异"上移为"**职业内不同 Build 手感差异**"——同一个亚马逊,玩家应能在 30 秒录像内分辨"散射割草"与"冰控精准"是两种打法。

| 体验轴 | 亚马逊(弓系远程爽感) |
|---|---|
| 力量幻想 | 战场压制:元素/穿透箭幕覆盖战场,怪堆走不到我面前就被点名清空 |
| 风险距离 | **25~40 码风筝**:靠冰冻箭控场、翻滚回避拉开,贴脸即危险 |
| 爽感反馈通道 | 箭矢弹道/穿透命中表现、多重射击扇形刷屏、冰冻碎裂死亡变体、暴击黄字、传奇光柱 |
| 资源心智 | **专注=回复—爆发型**:上限 100,被动回复 8/秒,利箭命中回 4;鼓励"3 秒爆发一轮→数秒走位回填" |
| 操作节奏 | 利箭点名产能(零消耗)→多重/冰箭一轮泄能→翻滚走位→女武神招牌大招协战;手离怪、眼不离场 |
| Demo 验证点 | 弓系弹道手感、穿透/控场价值、"**装备改变施法密度**"(专注引擎,见 5.2)、**职业内 Build 手感差异** |
| 盲测验收 | 蒙住技能图标,旁观者凭"什么时候能放技能"与屏幕反馈(散射刷屏 vs 冻场碎裂)即可分辨 Build(沿用正式版盲测支柱) |

> 单职业不等于"内容单薄":由于砍掉了第二职业的实现/调参/QA 预算,亚马逊得以保留**穿透箭幕、冰冻碎裂、女武神召唤、终极符文**等高表现度内容,把"弓系到底爽不爽"这件唯一要紧的事验证到位。

### 1.3 从正式版继承/删减的规则清单

| 类别 | 继承(不改) | 删减/简化(Demo 不做) |

## 4. 精选技能清单与 5 主动技能数值表

### 4.1 从正式版技能池挑选的五条标准

1. **链路完整**:产能 / 泄能(AoE) / 控场 / 位移 / 召唤增益,**五类在 5 个技能内一对一恰好全覆盖**——保证 5 槽永远装得出"健康配方",不浪费任何一槽。
2. **表现度优先**:Demo 首要任务是验证弓系手感,选屏幕反馈最强的技能(多重射击扇形刷屏、冰冻碎裂溅射、女武神协战),而非数值最优技能。
3. **实现成本可控**:符文优先选数值/换元素档(配置可生成),仅保留少量终极档(钢羽、寒霜碎裂、双姫)做"质变记忆点";位移技能仅 1 条符文。
4. **Build 支撑**:5 技能 + 符文必须能拼出 **2~3 套操作动词不同的弓 Build**(散射割草 / 冰控精准 / 机动游击),见第 6 章。
5. **排除项**:依赖终局套装或词缀才成立的技能、长 CD 召唤群等性能高风险技能、与 8 级数值框架脱节的超倍率技能一律不选。女武神作为亚马逊身份招牌保留,顺带验证"召唤协战增益"这一表现用例。

## 5. 专注资源循环设计:8 级框架下如何成立

正式版的资源循环在毕业期由装备词缀闭合(APoC 类:暴击回资源)。Demo 没有词缀深度,**把闭环责任从"随机掉落"转移到"解锁脚本 + 1 件固定剧本传奇"**,并保证专注"回复—爆发型"的身份依然成立。

### 5.1 专注:回复—爆发型(产耗测算,12 级框架→8 级框架)

| 参数 | 正式版(70 级) | Demo(V1.0,12 级框架) | Demo(V2.0,8 级框架) | 调整理由 |
|---|---|---|---|---|
| 上限/被动回复 | 100 / 高回 | 100 / 回复型 | **100 / 被动回复 8/秒** | 不动池容,回复定为 8/秒,匹配"数秒回填" |
| 实战回能 | 毕业靠 APoC 词缀 | 符文导能 + 资源被动 | **①利箭命中回 4;②被动 c 充沛精力 +3/秒、脱战不衰减;③固定剧本传奇「专注引擎」(暴击回 6)在屠夫战前(≈13~17 分钟)保底掉落** | 把"装备改变施法密度"压进流程内,作为掉落→构建循环的教学样本 |
| 死锁保险 | RES-05 | 继承 | **利箭零消耗,永远可按** | 专注耗尽也能继续利箭点名,不会站桩发呆 |

**产耗测算(5 级 5 槽成型时)**:开局满池 100 倾泻 → 多重射击 20 / 冰冻箭 25 一轮泄能 → 利箭命中回 4 + 被动 8/秒(拿到充沛精力后 11/秒)回填 → 形成"**3 秒爆发,数秒回填**"的呼吸节奏。

- **群战(≥4 怪)**:利箭多目标命中 + 多重射击命中数高,回能快,扇形可近似连发;
- **单体(屠夫)**:满池只够倾泻 2~3 轮泄能,之后被迫切回利箭点名攒池——这个"Boss 战呼吸感"是刻意保留的设计,屠夫战(目标 TTK 100~150 秒)的节奏检定即来源于此。

**装备改变手感的验证样本**:拿到传奇「**专注引擎**」(暴击回 6 专注)后,施法密度肉眼可感变化——泄能轮次间隔明显缩短,玩家应能明确说出"捡到那把护身符之后手感变了"。这是 V2 验证目标"装备改变手感"的核心样本(与 04-Demo 装备文档联合设计)。

## 6. Build 验证设计(梦魇挑战,2 层)

### 6.1 验证目标与难度对位

V2.0 验证命题为"**职业内不同 Build 手感差异是否成立**"+"掉落→构建→挑战更难内容的循环能否在 30 分钟内成立"。梦魇为 **2 层**,纯数值缩放、复用关卡 2(大教堂)/关卡 3(痛苦大厅)Tile,**不做新地图/新 Boss/新美术**。

| 梦魇层 | 复用关卡 | 强度缩放(以 Lv8 白怪 265/45 为 100% 基准) | 守门 | 通过预期 |
|---|---|---|---|---|
| 层1·苏醒 | 复用大教堂 Tile | **×1.5 生命 / ×1.25 伤害**(白怪绝对 398/56),密度+ | 黄名强化精英(复用,**非新 Boss**) | ≈75% 玩家推进至此;沿用通关屠夫的配置即可过 |
| 层2·核心 | 复用痛苦大厅 Tile | **×2.2 生命 / ×1.5 伤害**(白怪绝对 583/68) | **梦魇屠夫=屠夫数值×1.5≈16500**,烧地板间隔 18s→12s | ≈50% 通关;迫使针对性调整符文/被动 |

**锁定规则**:进入梦魇层瞬间快照技能/符文/被动(继承正式版大秘境锁定),层间可调整——"层间调 Build"本身就是验证目标 V3。**盲测验收**:旁观者观看 30 秒战斗录像,能分辨同职业不同 Build(动词不同:散射刷屏 / 冻场风筝 / 贴脸自爆输出)。

### 6.2 三套弓 Build(职业内构建差异样本)

| | A 多重散射(AoE 割草/风筝) | B 冰控精准(控场/Boss) | C 机动游击(高风险/操作,变体) |
|---|---|---|---|
| 左键 | 利箭·穿透箭 | 利箭 | 利箭·穿透箭 |
| 右键 | 多重射击·火力覆盖 | 冰冻箭·寒霜碎裂 | 多重射击·火力覆盖 |
| 键1~3 | 冰冻箭·深冬 / 翻滚·疾风 / 女武神·双姫(钢羽承载扫射感) | 女武神·战姫祝福 / 翻滚·疾风 / 多重射击·钢羽(万箭由钢羽) | 冰冻箭·深冬 / 翻滚·疾风 / 女武神·战姫祝福 |
| 被动 | 致命精准 + 充沛精力 | 致命精准 + 鹰眼 | 闪避本能 + 不屈意志 |
| 核心装备(04-Demo 文档定名) | 传奇弓「女妖之弓」:多重射击额外 +2 投射物 | 传奇箭袋「冰霜之箭袋」:冰冻/冻缓目标受到伤害 +30% + 传奇头「女武神之冠」:召唤数+1、持续+50% | 传奇靴「疾风之靴」:翻滚回避无 CD(改为消耗 25 专注),翻滚留诱饵 |
| 一句话循环 | 群战永动:穿透箭幕铺场,多重扇形割草,翻滚风筝 | 冻场风筝:冰冻箭锁场,寒霜碎裂连锁清群,女武神协战压 Boss | 走位贴脸输出:疾风之靴无 CD 翻滚,贴脸多重 + 冰箭自保,高风险高操作 |
| 验证点 | AoE 割草手感、群战专注永动、钢羽扫射的刷屏爽感 | 控场价值、冰冻碎裂连锁、单体/Boss DPS 检定、女武神召唤体验 | 机动性反差、"无 CD 翻滚"的操作上限、专注被翻滚抽走后的循环边界 |

> A/B 是两套"正轨"对位样本(散射 vs 冰控),C 是"玩法反差样本"(高机动高风险),不强求极限强度。三套核心件分别落在弓 / 箭袋+头 / 靴,确保"换装即换手感"在装备层有依据。

### 6.3 Build 层验收口径

1. **A/B 两套必须能通梦魇层 2**,C 套至少推进至层 2(C 套定位是"玩法反差样本",验证机动游击手感而非极限强度);
2. 任意两套 Build 的 5 槽**重合 ≤3 个技能,或重合技能符文不同**——保证"可感知差异"有配置层依据;
3. 平衡基线(对齐正式版口径收缩):同投入下任意两套可通层数差 ≤1 层;
4. **进梦魇前后调整技能/符文/装备的玩家比例 ≥60%**,全程主动换装中位 ≥4 次(对齐 V3 判定)。
