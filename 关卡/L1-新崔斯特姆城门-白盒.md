## 一、定位与规模(先说清"不做什么")

第一幕开场战斗区,约 **3–5 分钟**单段体验,验证**弓系核心手感 + 升级第一跳**(槽2 多重射击)。这是 **垂直切片的第一个可玩原型**,刻意压到"一条主路 + 一个环路支线 + 一场尸潮高潮"的最小可玩集。

**严格避免过度设计**:无解谜、无多分支剧情门、无脚本演出、无第二个 Boss;地标/资源用占位盒子,数值走系统② 既有 `SpawnManager`/`enemy_zombie`。规模 ≈ 一屏半,5 个房间矩形,17 只白怪(分 4 波,峰值并发 ≤12),契合 2 天工期里"关卡C Day1 战斗区1 灰盒(3h)"的预算。

## 二、借鉴的 D3 关卡设计原则 → 本关怎么落

| D3 原则 | 本关落地 |
|---|---|
| **清晰视觉引导 / 地标牵引** | 三级地标沿主路递进:广场**破井**(出生朝向锚点)→ **城门**(全关最强地标,门柱+门楣框出广场入口,远处即见)→ 出口外**大教堂剪影**(高耸暗块,牵引玩家朝 L2 出口前进)。玩家"看着地标走",无需小地图。 |
| **环路探索(非死胡同)** | 主街 B 到城门广场 D 有**两条路**:直走 B→D,或东拐进**资源支廊 C(宝箱)**再南下汇入 D。绕行被奖励(首把蓝弓),且不回头——这是 D3"探索有回报、动线不折返"的最小环。 |
| **节奏感:战斗区↔喘息交替** | 谷(教学)→ 小冲突(主街伏击)→ **峰1 尸潮高潮**(城门广场)→ **休整区**(篝火安全区,零刷怪)。峰谷落差明显,符合 `03 §9` 的"20–40s 战斗 + 10–20s 移动"脉冲。 |
| **地标节点 = 遭遇锚点** | 每个地标都是一次遭遇/决策:破井=教学战、城门=尸潮、宝箱=绕路决策、篝火=喘息+回血、出口=进度门。 |
| **中心掩体/绕怪空间** | 城门广场 18×16 开阔,玩家围绕广场周边**风筝**尸潮;门柱提供视觉掩体。给弓系"拉开—点名—泄能"留足走位空间。 |

## 三、核心玩法循环与玩家路径

**核心循环(本关验证的最小闭环)**:
`进入 → 看地标定向 → 遭遇白怪(走位+射击)→ 多重射击解锁/泄能割草 → 绕路捡装(变强)→ 城门尸潮高潮(运用走位+AoE)→ 篝火喘息 → 出口(进度兑现)`

**玩家路径(主路 S 形,北为前,玩家面朝 −Z)**:

```
出生(广场南) → ①教学战(3走尸) → 主街 → [分叉]
                                          ├─ 直走 → 城门广场
                                          └─ 东拐 → 支廊宝箱(首蓝弓) → 南下汇入城门广场
城门广场:②尸潮(8环绕)+ ③增援(4从门涌入) → 清场
        → 穿城门北上 → 篝火休整(回血/喘息) → 出口(→ 大教堂 L2)
```

主路单向北进(不折返);唯一支线(支廊)是"绕一下拿奖励再回到主路"的环,不打断节奏。

## 四、区域规划:战斗 / 休整 / 资源

| 区 | 名称 | 世界坐标 (X,Z) | 类型 | 内容 |
|---|---|---|---|---|
| A | 入口广场 | X[-7,7] Z[16,26] | 教学战 + 出生 | 玩家出生(0,0,24);**破井**地标;**①** 3 走尸(cluster)教"移动+射击,利箭 2 发死" |
| B | 主街 | X[-4,4] Z[6,16] | 过渡 + 小冲突 | **②** 2 走尸(line)轻伏击;环路分叉口 |
| C | 资源支廊 | X[4,11] Z[6,14] | **资源点** | **宝箱**(脚本掉首把蓝弓 DPS+60%);绕路奖励;南口汇入城门广场 |
| D | 城门广场 | X[-9,9] Z[-10,6] | **战斗高潮(峰1)** | **城门**地标;**③** 8 走尸 surround(尸潮)+ **④** 4 走尸从城门增援;18×16 风筝场 |
| E | 休整出口 | X[-5,5] Z[-18,-10] | **休整 + 进度门** | **篝火**安全区(零刷怪,治疗锚点);**出口传送门**→ L2;外有大教堂剪影 |

- **战斗区**:A(教学)、B(小)、D(高潮)。**休整区**:E(篝火,刻意无怪)。**资源点**:C(宝箱)+ 全程罐桶位(白盒先省,Day2 补金币/血球罐)。
- 死亡压力对齐 `05 §7.2`:主线刻意宽松,白怪 12–18 击致死;尸潮靠"数量包围"制造张力而非高伤害。
